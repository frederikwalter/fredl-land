Some thoughts about the programming languages I am learning:

I kinda like C. Its simple to understand and there are not a lot of complicated instructions to remember. It all comes down to the logic, you apply. However, caring about memory and shit is complicated as well.

I am not sure about Python yet. Everybody uses it, it is easy to get started and there are a lot of libraries available such that I can rely on other peoples work. I think some experience is required to get a feeling for available libraries and how to use them.

It is quite fun to play around with SQL. Maybe the awesome riddle with the detective work played a role, but it is quite easy to get into SQL and compile complex queries.

I am not a fan. So many different commands and functions already built in. It is hard to understand it in full what is happening and then it comes down to copy and paste a lot. And there are no error messages, which sucks.
